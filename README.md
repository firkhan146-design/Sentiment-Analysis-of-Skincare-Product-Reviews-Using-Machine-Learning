# Sentiment-Analysis-of-Skincare-Product-Reviews-Using-Machine-Learning
📌 Project Overview

This project uses Natural Language Processing (NLP) and Machine Learning to analyze customer reviews of skincare products and classify them into positive or negative sentiments.
The goal is to help brands understand customer feedback, improve products, and make data-driven marketing decisions.

🎯 Problem Statement

Skincare brands receive thousands of online reviews, making manual analysis difficult.

This project aims to:

Automatically analyze customer opinions
Identify satisfaction trends
Extract insights from textual feedback

🤖 Machine Learning Models Used
Naive Bayes
Support Vector Machine (SVM)

📊 Key Features

✔ Text preprocessing (stopwords removal, stemming, tokenization)
✔ TF-IDF feature extraction
✔ Model comparison and evaluation
✔ Automated sentiment classification

💡 Applications
Understand customer satisfaction
Improve skincare products
Support marketing & product strategy
